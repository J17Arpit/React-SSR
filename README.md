# React-SSR
SSR React from scratch POC

1. cd React-SSR
2. yarn
3. yarn start

#Following steps were done to built it.
1. This app is built on 'create-react-app'
2. Later on, webpack support and server side rendering was added.

#Final output to be used from 'server-build'

#Approach followed during design
Mobile first approach was followed and later following media queries were used to handle the design for tablet and desktop.

1. @media only screen and (min-width: 700px) 
2. @media only screen and (min-width: 1024px)

#Tool used -> WEBPACK
To generate the production optimized build and enable server side rendering.
